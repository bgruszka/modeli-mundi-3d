Historical Models Included

Aristotle's Model (384-322 BC)

Earth-centered universe with crystalline spheres carrying celestial bodies.

Ptolemaic Model (150 AD)

Sophisticated geocentric system using epicycles to explain planetary motion.

Copernican Model (1543)

Revolutionary heliocentric model with the Sun at the center.

Galilean Model (1610)

Enhanced heliocentric model featuring Jupiter's four largest moons.

Kepler's Model (1609)

Most accurate historical model with elliptical planetary orbits.
